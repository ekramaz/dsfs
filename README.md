    刘涛见王天开始冲自己来了，赶紧扯开话题，再让大哥说下去，几个小时是不会完了：[大哥，你说我们招齐人后，是不是再干他几票，多搞点钱。]

    王天沉默了，有点懊恼，良久，开口说道：[现在先别做，钱还够花，这段时间，我一直在想一个问题，我这样做对不对？]掏出烟，点上一只：[不管我怎么恨世嫉俗，不管我怎么觉得对我不公平，但是，我毕竟是一个中国人，这一点是改不掉的，这次我们运的货，销到了全国各地，也就是在变相的毒害自己人，我有点不忍心，也许我还有那么一点点爱国心吧。]苦笑一声，[一个贩了三吨毒品的人会爱国，说了有谁会信。我他妈的自己不敢相信我自己。不管怎么样，害自己国人的事，能少做尽量少做，为了我自己那颗渺小的良心吧。]

    两兄弟不禁也沉默起来。罗放是什么都听王天的，从小跟着王天，现在就服他一个，反正大哥说不做，自己就不做，听大哥的准没错。

    刘涛就比罗放有想法，毕竟受过三年警察教育，还当了一年警察，爱国情结还是比较浓的，只是当时没有转过弯来，现在听王天一讲，也觉得有些过份，挺没劲的，害自己人算什么，有本事出去害老外，对老外也来一次鸦片战争，那才叫为国争光。

    王天大呼了一口气：[不说这个了，三弟，你明天也去办件事，给我把我们的邻居青狼的情况调查清楚。]刘涛知道王天要开始报仇了，兴奋的点点头。

    事情交待完了，王天去拿了几块木板，拉着不情不愿的两人去了沼泽。王天先在五十米处放上了两块木板，让罗放和刘涛坐上去，交待道：[你们就在这里练，撑不住了就往回走一点，练够两个小时就可以了。]

    罗放问道：[大哥，那你呢。][不用管我，我到里面去，你们练够时间就先回去，知道吗？]

    王天见两兄弟都明白了，自己就向前走去，走到六十米处，拔出刀，又砍了十米，到了七十米处。这里的冷气已十分重了。王天拿出木板，盘腿坐了上去。

    小天果然没有说错，这里真是个练功的好地方，原来都是一丝一丝的吸收，现在是一股一股的。深吸一口气，王天沉下心，不断推动心核，强大的能量涌入了心核，核内马上反旋转，把自己的身体包住。王天感觉到能量在自己身体进进出出，锤炼着自己的心肝脾肺肾，不断加强它们的机能。还有能量沿着自己的经脉，从心核流出，走了一遍后，又回到心核，周而复始。

    不知过了多久，王天身体一振，心核已长到弹珠大小，旋转的力量更多，流出的能量也更强了。他知道,已进入第二层.

    王天把意识伸进了异空间，脑袋炸了一下，好久才回过神来，这就是小天吗？这就是一仙女嘛，一双乌黑的眼睛明亮剔透，仿佛会说话，眼角还有丝泪痕，楚楚动人，一对弯弯的柳叶眉，小巧精致的鼻子下面一张小嘴樱桃般红润欲滴，黑亮的长发如瀑布般垂下，所有的赞美语言放在她身上都显得那么的无力，上苍好象把天地所有的灵气都汇到了她身上。等一下，怎么越看越眼熟？好象在哪儿见过。不可能，这样的美女自己见过一次一定不会忘记，这是怎么回事？

    小天看见王天的傻样，害羞的道：[我是按你心中最完美的美女形象生成的，怎么样，好不好看？]说完，向王天嫣然一笑。

    王天眼珠子都快掉出来了：[好看，好看。]下意识的回答：[啊，你，你，你说什么？]终于缓过神来。听小天又说了一遍，马上问道：[不会吧，你还可以看到我的隐私？完了完了，没脸见人了。]说着就去撞树。

    [不是啊，是原来的，现在已经看不到了。]言语透着一丝惋惜。

    [那还好，尊重隐私是每个人都应遵循的。]王天义正言词道。[屁，只是没有机会。]

    [天少，可不可以求你一件事，]小天小心翼翼的说。[没问题，说吧。]王天已经被迷昏了头脑。

    [就是那个阴阳玉，我无法吸收能量，可不可以给我一块。]

    [一块？难道有几块吗？][不是的，它可以分成好几块的。]

    [那行，等我拿到了就给你。]王天开着空头支票。

    [谢谢。]小天向王天道谢。

    [我的能量不够了，必须回去了，只有有了能量我才能再出来。]小天擦了擦眼睛，开始消失。

    小天好像又想了什么似的说：[还有，我已经把心法的其他部分存到了笔记本里，我形成意识后是要消耗能量的，现在你还可以隐形十分钟。]说完，吐了吐舌头，消失了。

    王天苦笑了一下，一下就用掉了十三分钟，可真够浪费的。又轻声叫道：[小天？小天？]没有人应答。王天知道小天是彻底没能量了，暗暗下定决心一定要好好练功，把那块破石头早点取出来。

    看了看天，已经亮了，赶紧出去了，发现罗放和刘涛两人在沼泽的入口睡着了，感动的叫醒他们。

    [大哥，你出来了？怎么这么久？]罗放一边打哈欠一边说道。猛的发现大哥有了一点不同，看了看刘涛，他也在瞪着大哥。

    刘涛把王天仔细打量，对，飘逸，大哥的气质又增加了不少，小声的问：[大哥，功力高了？]王天笑着打了他一下：[是的，增加了一点，你们呢？]拿起两人的手腕，用心神查探，嗯，不错，一晚上两人的进步都挺大的，虽然距心核还有段距离，但前景很好，游走的气还是很足的。

    吩咐两人去办昨天定下的事，自己也开车出去了。

    王天想了想，事还挺多的，当务之急是请一个施工队，把自己的地给围起来，沼泽的密秘是一定要守住的，要想个办法让别人看不出来。嗯，还有罗放去找人，没武器不行啊，找谁买呢？老鼠？不行，一支两支可以，多了他也没办法，还是找刘敬元吧，等一下就给他打电话。青狼的事，也该处理了，先要给郭玉龙打个招呼，免得他借题发挥。

    想好后，就一通电话打给了刘敬元：[老哥，是我，王天。]

    [是老弟啊，到家了吧？]刘敬元的声音传出。

    [到了，谢谢老哥关心，有件事，我想麻烦你老哥。]

    [没说的，什么事只管开口。]刘敬元打包票。

    [我们想买点军火，老哥你有没有什么路子，指点我一下。]

    刘敬元想了想，问道：[要多少，多的话我给你发过去，少的话，你就直接找郭玉龙，他有。]

    [要的也不多，二十把手枪，二十把冲锋枪，外加两支狙击，也就够了。]

    刘敬元咂了咂舌。武京那个地方，一个帮派才多少人，郭玉龙全部人马也就六百多，才有十几把手枪，这次还是我卖给他得了：[郭玉龙那里估计没这么多，要不这样，我给你发过去，手枪每把配200发子弹，冲锋枪配500发，狙击200发，全部是顶级的德国货，一百三十万美金，怎么样？]

    王天算了一下，同意了。
